Notas Tecnicas

Decisoes e limitacoes do script

🌐

DNS Google

O script configura automaticamente o DNS para 8.8.8.8 / 8.8.4.4 em todos os adaptadores ativos antes de comecar.

📦

WinRAR via Release

Winget dava erro na instalacao do WinRAR, entao o instalador e baixado diretamente da release do GitHub.

👁

AnyDesk Manual

Precisa de clique manual para instalar como servico. O script abre o instalador e espera voce fechar.

⚠

Barra Win 11

Fixar programas na barra de tarefas via script nao funciona de forma confiavel no Windows 11. Fazer manualmente.

📂

Explorer nao reinicia

O Explorer nao e reiniciado no final para evitar abrir uma janela indesejada. Mudancas aplicam no proximo reinicio.

❌

Fusion 360 removido

Removido do script pois tem atualizacoes frequentes e a licenca pessoal precisa ser baixada na hora.
