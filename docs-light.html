Formatei o PC e Agora o Que Fazer?

Uso: Pessoal  |  Ultima atualizacao: Marco 2026  |  Repositorio: github.com/GCintra00/formatei-pc

Compatibilidade: Windows 10 e Windows 11

1. Como Executar

Script que configura o PC apos uma formatacao: remove bloatware, instala programas, personaliza o desktop e limpa o sistema.

Pre-requisitos

  • PC conectado a internet
  • Desktop do Windows acessivel (OOBE finalizado)

Passos

1 Abrir o PowerShell como Administrador

Clique com botao direito no icone do Windows > Terminal (Admin) ou Windows PowerShell (Admin)
2 Colar e executar o comando abaixo:
Get-NetAdapter | Where-Object {$_.Status -eq 'Up'} | ForEach-Object { Set-DnsClientServerAddress -InterfaceIndex $_.ifIndex -ServerAddresses ("8.8.8.8","8.8.4.4") }; irm https://raw.githubusercontent.com/GCintra00/formatei-pc/master/setup.ps1 | iex
3 Aguardar a execucao. Ha dois passos manuais:
  • AnyDesk: Clicar em "Instalar AnyDesk" quando a janela abrir
  • uTorrent Web: Clicar Next > Agree > Skip All > Finish
DNS automatico: O prefixo do comando configura o DNS do Google (8.8.8.8) antes de baixar o script, garantindo que o download funcione mesmo com DNS lento.

Versao Light (so Chrome + limpezas)

1 Abrir o PowerShell como Administrador
2 Colar e executar:
Get-NetAdapter | Where-Object {$_.Status -eq 'Up'} | ForEach-Object { Set-DnsClientServerAddress -InterfaceIndex $_.ifIndex -ServerAddresses ("8.8.8.8","8.8.4.4") }; irm https://raw.githubusercontent.com/GCintra00/formatei-pc/master/setup-light.ps1 | iex
Diferenca: A versao light instala apenas o Google Chrome, mas executa todas as limpezas (bloatware, notificacoes, barra de tarefas, auto-inicio). Nenhum passo manual necessario.

5. Auto-Inicio (Startup)

O script remove todos os programas do auto-inicio, mantendo apenas:

ProgramaMotivo
AnyDesk (--control)Acesso remoto em segundo plano
LightshotScreenshots rapidos
SecurityHealthWindows Defender
RtkAudUServiceDriver de audio Realtek

A limpeza e feita em multiplas camadas:

  • Registro HKCU\...\Run e HKLM\...\Run
  • Pastas Startup (usuario e comum)
  • StartupApproved (Gerenciador de Tarefas)
  • Tarefas agendadas de logon
  • Configs especificas: Discord (settings.json), Steam, Slack, Epic Games, Logitech G Hub

7. Solucao de Problemas

ProblemaSolucao
"Execute como Administrador"Feche o PowerShell e abra com botao direito > Executar como Administrador
Erro de DNS / download falhaO script ja corrige o DNS automaticamente. Se persistir, conecte via cabo.
AnyDesk nao instalaO script abre o instalador. Clique em "Instalar AnyDesk" na janela que abrir e feche depois.
Programa deu ERROExecute o setup novamente — programas ja instalados serao detectados e pulados.
Barra de tarefas nao fixou programasLimitacao do Windows 11. Fixar Opera GX, Chrome e Explorador manualmente.
Mudancas na barra nao apareceramReinicie o PC. O Explorer nao e reiniciado pelo script para evitar janelas indesejadas.

8. Manutencao do Script

Repositorio: https://github.com/GCintra00/formatei-pc
Qualquer alteracao feita no repositorio e aplicada automaticamente na proxima execucao do comando.

Arquivos

ArquivoFuncao
setup.ps1Script principal de configuracao (8 etapas)
Release v1.0Instaladores e arquivos extras (12 arquivos)

Para adicionar/remover programas Winget

Editar o array $programas no setup.ps1. O ID pode ser encontrado com winget search "nome".

Para atualizar instaladores da Release

Criar uma nova Release no GitHub com os novos arquivos e atualizar a variavel $ghRelease no script.
